\begin{nums}{-2}
\item Формулы алгебры логики. Существенные и несущественные
переменные. Формулы. Эквивалентность формул. Элементарные функции
и их свойства. Разложение функция по переменной. Совершенная
дизъюнктивная нормальная форма (СДНФ).
\item Полные системы функций. Примеры полных систем. Замкнутые классы.
\item Полиномы Жегалкина. Линейные функции. Лемма о линейной функции.
\item Самодвойственные функции. Лемма о несамодвойственной функции.
\item Монотонные функции. Лемма о немонотонной функции.
\item Теорема о полноте систем функций алгебры логики. Возможность
выделить из каждой полной системы полную систему, состоящую не
более чем из четырех функций. Предполные классы.
\item Функции $k$-значной логики. Элементарные функции. Полнота системы
$$\{0, 1,\ldots, k-1, J_0(x),J_1(x),\ldots, J_{k-1}(x),\max(x, y),\min(x, y)\}.$$
\item Полнота систем $\{\max (x, y), x+1\}$, $\{V_k(x, y)\}$.
\item Алгоритм распознавания полноты конечных систем функций в $P_k$.
\item Представление функций из $P_k$ полиномами.
\item Лемма о трех наборах. Лемма о квадрате. Теорема Яблонского.
Теорема Слупецкого.
\item Классы сохранения множеств функций и их свойства. Теорема Кузнецова.
\item Замкнутый класс в $P_k\ (k\ge 3)$ без базиса. Замкнутый класс
в $P_k\ (k\ge 3)$, имеющий счетный базис. Мощность множества
замкнутых классов в $P_k$.
\item Графы, основные понятия. Реализация графов в трехмерном
пространстве.
\item Схемы из функциональных элементов. Простейшие методы синтеза.
\item Контактные схемы. Простейшие методы синтеза.
\item Метод каскадов (для контактных схем и функциональных
элементов).
\item Верхняя оценка функции $L_n$ для схем из функциональных
элементов и контактных схем. Порядок роста функции $L_n$.
\item Детерминированные функции. Ограниченно-детерминированные
функции (автоматные) функции. Способы их задания.
\item Реализуемость автоматных функция схемами из функциональных
элементов и элементов задержки (схемами с обратными связями).
\item Исчисление высказываний. Аксиомы. Правило вывода.
Тождественная истинность выводимых формул. Непротиворечивость
исчисления высказываний.
\item Предикаты. Логические операции над предикатами.
\item Условие полноты системы предикатов на конечном множестве.
\item Модель, формулы в модели. Свободные и связные переменные.
Значение формул в модели.
\item Истинность формул в модели, на множестве. Тождественно
истинные формулы.
\item Правила эквивалентных преобразований формул логики предикатов.
Нормальная форма.
\item Машина Тьюринга. Вычислимые функции.
\item Алгоритмическая неразрешимость проблемы самоприменимости.
\item Алгоритмическая неразрешимость проблемы применимости.
\end{nums}
