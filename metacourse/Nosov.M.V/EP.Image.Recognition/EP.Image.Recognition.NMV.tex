\begin{nums}{0}
\item Постановка задачи распознавания. Пример алгоритма распознавания. Материал обучения.
\item Понятие теста, тупикового теста. Утверждения о тестах материала обучения,
таблицы сравнения и приведённые таблицы сравнения.
\item Лемма о структуре множества тестов. Алгоритм отыскания множества определения тестов.
\item Понятие информационного веса признака. Линейные процедуры распознавания (алгоритмы $А_1$, $А_2$, $А_3$).
\item Понятие асимптотически правильно распознающего линейного алгоритма. Формулировка теоремы существования
лемма о существовании булевской матрицы с заданным спектром. 
\item Понятие пороговой функции. Нижняя оценка числа пороговых функций. Вектор Чоу. Теорема о единственности
вектора Чоу. Верхняя оценка числа пороговых функций.
\item Лемма Шпернера. Лемма Литтлвуда\ч Оффорда. Ослабленный вариант леммы Одлыжко. Нижняя оценка числа
пороговых функций. 
\item Понятие асимптотически правильно распознающего линейного алгоритма. Теорема существования
(доказательство без вспомогательной леммы). 
\item Понятие теста. Алгоритм голосования по тестам. Пример. Многочлен голосования. Разложение многочлена
голосования по ортонормированному базису. Приближения многочлена голосования.
\item Понятие $М_k$\д пороговой функции. Утверждение о существовании многочлена с заданными значениями
в вершинах единичного куба. Утверждение о полиномиальном представлении линейной булевской функции.
\item Теорема о степени разделяющего полинома монотонной булевской функции.
\item Лемма Нечипорука. Нижняя и верхняя оценки числа $М_k$\д пороговых функций.
\end{nums}
