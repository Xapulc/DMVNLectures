\section*{Программа экзамена}

\begin{nums}{-2}
\item Основные понятия квантовой механики: наблюдаемые, состояния, чистые состояния, среднее значение наблюдаемой.
\item Соотношение неопределённости Гейзенберга. Амплитуда вероятности перехода.
\item Оператор эволюции квантовомеханической системы. Представления Шрёдингера и~Гейзенберга.
\item Квантовый гармонический осциллятор. Собственные значения оператора энергии, собственные состояния, пространство Фока \итп
      Координатное представление.
\item Группа Лоренца, касательное пространство в~единице группы Лоренца (то есть её алгебра Ли). Матричная экспонента.
\item Классические поля. Лагранжев формализм, уравнение Эйлера\ч Лагранжа. Теорема Нётер.
\item Вещественное скалярное поле. Лагранжиан. Уравнение Клейна\ч Гордона\ч Фока. Его решение.
\item Тензоры энергии\д импульса, момента\д импульса для вещественного скалярного поля.
      Выражение через полевую функцию и~функции~$a^\pm(\bfk)$.
\item Тензор момента\д импульса вещественного скалярного поля.
\item Комплексное скалярное поле. Лагранжиан. Уравнение. Решение.
\item Выражения заряда комплексного скалярного поля через полевую функцию и~через функции $a^\pm(\bfk)$ и~$\overset*a{}^\pm(\bfk)$.
\item Алгебра Клиффорда. Определение, основные примеры.
\item Связь групп $\Pin_{p,q}$ и~$\Spin_{p,q}$ с~группами $\Ob_{p,q}$ и~$\SO_{p,q}$ соответственно.
      Пример пути в~$\Pin_{1,3}$, соединяющего~$\pm 1$.
\item Разложение представления группы $\Pin_{1,3}$ в~пространстве $\Cl_{1,3}$ в~сумму четырёх подпредставлений.
      Доказательство того, что размерности этих представлений равны $4$.
\item Спинорное представление, спинорное поле, уравнение Дирака, сопряжённый спинор, лагранжиан.
\item Решение уравнения Дирака.
\item Выражение заряда и тензора энергии\д импульса спинорного поля через полевую функцию.
\item Квантование полей, квантование свободных полей. Уравнение эволюции (движения)
      вещественного скалярного поля.
\item Фоковское представление для вещественного скалярного поля.
\end{nums}

\section*{Задачи}

\begin{nums}{-2}
\item $(*)$ Пусть $\om$\т состояние. Тогда существует линейный положительный самосопряжённый оператор $\rho_\om$,
      такой что $\tr\rho_\om = 1$ и для всех $A \in \mathfrak{a}$ имеем $\ha{A}_\om = \tr(\rho_\om A)$.
\item Доказать, что $\tr A$ не зависит от выбора базиса.
\item Доказать, что если $\ph\cln M^4 \ra M^4$\т диффеоморфизм, сохраняющий метрику,
      то $\ph$ лежит в группе Пуанкаре.
\item Доказать, что если $\rho_\om^s$\т проектор при $t = t_0$, то $\rho_\om^s(t)$\т проектор при всех $t$.
\item В пространстве всех матриц $M_4(\R) \cong \R^{16}$  группа Лоренца является подмногообразием.
\item Проверить, верно ли, что если $\rho_\om^s(t)$\т проектор на одномерное направление
(то есть чистое состояние), то $\rho_\om^s(t)$ является чистым состоянием для всех $t$.
\item Может ли самосопряжённый оператор не иметь собственных векторов?
\item Найти матрицы $P$, $H$, $Q$, $a^+$, $a^-$ в базисе $\hc{\psi_i}$ ($i\in \N$).
\item Найти все чистые состояния, в которых для данной наблюдаемой $A$ выполняется равенство $\De_\psi^2 A = 0$.
\item Проверить, что квантовомеханический оператор эволюции унитарен.
\item Получить уравнение $\frac{d}{dx}\psi = -\frac{i}{h} H\psi$.
\item Доказать равенство $\ha{A^S}_{\psi(t)_S} = \ha{A(t)^H}_{\psi_H}$.
\item Показать, что формула
      $$\psi(q) \rightsquigarrow \ol{\psi}(p) = \frac{1}{\sqrt{2\pi\hbar}}\int \psi(q) e^{-\frac{i}{\hbar}pq}dq$$
      задаёт изоморфизм между координатным и импульсным представлениями коммутационных соотношений.
\item Полиномы Эрмита определяются формулой
      $$P_n(x) = \suml{j=0}{\hs{\frac{n}{2}}}(-1)^j c_nj x^{n-2j}, \quad c_nj = \frac{n!}{2^j j!(n-2j)!}.$$
      Доказать:\\
      а) $\hr{x - \frac{d}{dx}} P_n(x) = P_{n+1}(x)$;\\
      б) $\psi_n(x) = P_n\hr{x\sqrt{2\om}}\exp\hr{-\frac{\om}{2}x^2}$;\\
      в) полноту системы функций $\psi_n(x)$ в $L^2(\R)$.
\item Вывести уравнение Эйлера\ч Лагранжа для действия
      $$S[u] = \int \Lc\br{u(x), \pd_\mu u(x)}\,d^4x.$$
\item Проверить, что если для однопараметрической группы преобразований выполняется равенство
      $D\Lc = \pd_\mu f^\mu$, то действие $S[u]$ инвариантно.
\item Проверить, что уравнение Клейна\ч Гордона\ч Фока инвариантно относительно группы Пуанкаре.
\item Доказать равенство
      $$\pf{}{x^0}\int d^3\bfx x^s T^{00} = P^s$$
      для вещественного скалярного поля.
\item Доказать для комплексного скалярного поля соотношение $\br{a^\pm(\bfk)}^* = (a^*)^\pm(\bfk)$.
\item Вывести для компонент $T^{0\mu}$ тензора энергии\д импульса вещественного скалярного поля выражение
      через полевую функцию.
\item Вывести для компонент $T^{0\mu}$ тензора энергии\д импульса комплексного скалярного поля выражение
      через полевую функцию.
\item Вывести для заряда комплексного скалярного поля выражение через полевую функцию.
\item Для вектора энергии\д импульса вещественного скалярного поля получить выражение через $a^\pm(\bfk)$.
\item Для вектора энергии\д импульса комплексного скалярного поля получить выражение
      через $a^\pm(\bfk)$ и $\overset*a{}^\pm(\bfk)$.
\item Для заряда комплексного скалярного поля получить выражение
      через $a^\pm(\bfk)$ и $\overset*a{}^\pm(\bfk)$.
\item Для вектора энергии\д импульса спинорного поля получить выражение
      через $a^\pm_r(\bfk)$ и $\overset*a{}^\pm_r(\bfk)$.
\item Для заряда спинорного поля получить выражение через $a_r^\pm(\bfk)$ и $\overset*a{}^\pm_r(\bfk)$.
\item Найти размерность алгебры Клиффорда как векторного пространства над $\R$.
\item Доказать, что $\Cl_{1,1} = \Mat(2,\R)$.
\item Доказать, что $\Cl_{0,2} = \Mat(2,\R)$.
\item Доказать равенство $v_1v_2 + v_2v_1 = -2\ha{v_1,v_2}$.
\item Доказать, что отображение $\al\cln\Cl_{p,q} \ra \Cl_{p,q}$, заданное на образующих формулой
      $$\al(e_k) = -e_k, \quad \al(1) = 1,$$
      корректно продолжается до гомоморфизма
      алгебры Клиффорда в себя; доказать, что продолжение однозначно;
      доказать, что это продолжение\т изоморфизм.
\item В пространстве $\R^2$ с нормой $x^2 - y^2$ (то есть в пространстве $\R^{1,1}$) задано
      линейное преобразование с~матрицей
      $$\rbmat{\ch \chi & \sh \chi\\ \sh\chi & \ch\chi}.$$
      Представить его в виде композиции двух отражений.
\item Доказать, что $\Cl_{p,q} = \Cl_{p,q}^0 \oplus \Cl_{p,q}^1$, причём $\Cl_{p,q}^i\Cl_{p,q}^j \subs \Cl_{p,q}^{i+j \pmod 2}$
      (то есть по верхнему индексу имеется $\Z_2$\д градуировка).
\item В доказательстве того, что ядро гомоморфизма $A\cln \Pin_{p,q} \ra \Ob_{p,q}$ совпадает с $\pm 1$,
      использовалось представление элемента, лежащего в ядре, в виде суммы чётного и нечётного элементов: $x = x^0 + x^1$.
      Доказать, что $x^1 = 0$.
\item Доказать, что $f_k^2 = f_k$, $f_jf_k = 0$ при $j \ne k$ и $f_1 + f_2 + f_3 + f_4 = 1$.
\item Проверить, что в $W_i$ нет меньшего инвариантного подпространства (то есть $W_i$\т неприводимый модуль).
\item Проверить, что представления в $W_i$ ($i = 1\sco 4$) эквивалентны.
\item Найти матрицы операторов $R_{e_i}$ в базисе пространства $W_1$, состоящем из векторов $f_1, e_0f_1, e_2f_1, e_0e_2f_1$.
\item Доказать, что если известно, что существует представление $R$ группы $\Pin_{1,3}$, а операторы
      $R_{e_k}$ ($k = 0\sco 3$) известны, то все операторы представления однозначно восстанавливаются.
\item Проверить, что лагранжиан спинорного поля инвариантен относительно группы Пуанкаре.
\item Выразить тензор момента\ч импульса  спинорного поля через полевую функцию. Указать в полученном выражении
      орбитальную и спинорную части.
\item Вывести из уравнения Дирака уравнения на $\chi^\pm(\bfk)$ и $\overset*\chi{}^\pm(\bfk)$.
\item Показать, что условия нормировки
      $$\overset*v{}^\pm_s(\bfk)v_r^\mp(\bfk) = \de_{sr}, \qquad v_s^\pm(\bfk)v_r^\mp(-\bfk) = 0$$
      однозначно задают базис пространства решений уравнений Дирака (после преобразования Фурье).
\item Получить условие нормировки решений, сопряжённых по Дираку:
      $$\overset*{\ol v}{}^\pm_s(\bfk)\overset*v{}^\mp_r(\bfk) = \pm\frac{m}{k_0}\de_{sr}.$$
\end{nums}
