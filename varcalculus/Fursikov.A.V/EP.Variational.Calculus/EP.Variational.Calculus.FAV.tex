\item Теорема о среднем. Следствие о непрерывной дифференцируемости.
\item Оператор Немыцкого и его дифференцируемость.
\item Теорема Люстерника.
\item Теорема о касательном пространстве.

\item Выпуклые экстремальные задачи. Теорема Куна\ч Таккера.
\item Задачи Лагранжа и оптимального управления: основные определения. Формальный вывод принципа максимума из принципа Лагранжа.
\item Доказательство принципа максимума для задачи оптимального управления со свободным концом.
\item Полунепрерывность снизу: эквивалентность трех определений.
\item Теорема Вейерштрасса о существовании точки минимума.
\item Принцип компактности. Общая теорема о существовании точки минимума.
\item Теорема Мазура. Следствия: 1) Выпуклое и замкнутое множество секвенциально слабо замкнуто.
      2) Выпуклая и полунепрерывная снизу функция полунепрерывна снизу относительно слабой сходимости.
\item Пространства Соболева для функций одного переменного. Теорема о сужении на
      граничную точку отрезка.
\item Пример Больца о не существовании решения вариационной задачи. Существенность условия полунепрерывности
      снизу относительно слабой сходимости в теореме существования.
\item Пример Вейерштрасса о несуществовании решения вариационной задачи. Существенность условия
      секвенциальной слабой замкнутости множества ограничений в теореме существования.
\item Пример гармонического осциллятора о не существовании решения вариационной задачи. Существенность
      условия коэрцитивности задачи в теореме существования.
\item Пространства Соболева функций многих переменных: определение, доказательство полноты.
\item Теорема о плотных множествах пространств Соболева (формулировка). Метод
      замыкания на примере определения сужения функции на границу области.
\item Теорема о рефлексивности пространств Соболева.
\item Многомерная вариационная задача: условие роста и проверка коэрцитивности и ограниченности снизу.
\item Квазирегулярные вариационные задачи, и проверка полунепрерывности снизу
      относительно слабой сходимости соответствующих функционалов.
\item Теорема Тонелли о существовании решения вариационной задачи.
\item Вариационные неравенства. Задача с препятствием.
\item Пространство $H^{-1}$. Существование единственного обобщенного решения задачи Дирихле для оператора Лапласа.
