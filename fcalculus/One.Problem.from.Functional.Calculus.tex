Итак, пусть $f$ бесконечно гладкая на прямой, $f(x)=0$ при $x\le 0$, $f^{(n)}(x)>0$ при $x>0$. Получим
противоречие. Без ограничения общности $f(1)=1$. Заметим, что мы можем писать формулу Тейлора с
остаточным членом в форме Лагранжа.

Запишем этот формулу для $x=1$ с центром в $0$:
$$1=f(1)=\sum_{i=0}^{n-1} \frac{f^{(i)}(0)}{i!} + \frac{f^{(n)}(\xi)}{n!} \le 
\frac{f^{(n)}(\xi)}{n!} \le \frac{f^{(n)}(1)}{n!}.$$
Отсюда $f^{(n)}(1) \ge n!$.

Запишем формулу для $x=2$ с центром в $1$:
$$f(2)=\sum_{i=0}^{n-1} \frac{f^{(i)}(1)}{i!} + \frac{f^n(\xi)}{n!} \ge \sum_{i=0}^{n-1} \frac{f^{(i)}(1)}{i!} \ge n$$
для любого $n$.
Противоречие.
