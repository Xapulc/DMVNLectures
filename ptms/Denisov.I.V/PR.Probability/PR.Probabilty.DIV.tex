\begin{problem}
\label{firstproblem}
Подбрасываются три игральных кости. Найти вероятность того, что выпало <<6>>, <<6>>, <<6>>, если известно, что на двух костях выпало <<6>>.
\end{problem}
\begin{solution}
Пусть $A$~--- событие <<выпало три шестёрки>>, а $B$~--- событие <<выпало две шестёрки>>. Вычислим условную вероятность по определению,
используя формулу $P(A|B) = \frac{P(A\cap B)}{P(B)}$. Очевидно, $P(A\cup B) = P(A)$, поскольку $A \subs B$. Очевидно, $P(A) = \frac{1}{6\cdot6\cdot6}$.
Не менее очевидно, что $P(B) = \frac{1\cdot1\cdot6 + 1\cdot6\cdot1 + 6\cdot1\cdot1 - 2}{6\cdot6\cdot6}$ (двойку пришлось вычесть, поскольку
мы посчитали исход <<666>> два лишних раза). Отсюда получаем ответ $P(A|B) = \frac{1}{16}$.
\end{solution}

\begin{problem}
Пусть $\xi, \eta$~--- случайные величины с функциями распределения $F(x)$ и $G(x)$ соответственно. Случайная величина $\zeta$ равна 1 с вероятностью $p$
и 0 с вероятностью $1-p$. Все они независимы между собой. Найти функцию распределения $\mu := \xi\zeta + (1-\zeta)\max\hc{\xi,\eta}$.
\end{problem}
\begin{solution}
Пусть $H(x)$~--- функция распределения $\mu$. Очевидно,
$$H(x) = \Pb\bc{\zeta\xi + (1-\zeta)\max\hc{\xi,\eta} \le x} = \Pb\bc{(1-\zeta)\max\hc{\xi,\eta} \le x,\; \zeta=1} +
\Pb\bc{(1-\zeta)\max\hc{\xi,\eta} \le x,\; \zeta = 0}=$$
$$=\Pb\bc{\xi \le x,\; \zeta = 1} + \Pb\bc{\max\hc{\xi,\eta} \le x,\; \zeta = 0} = p\cdot\Pb\bc{\xi \le x} + (1-p)\cdot\Pb\bc{\max\hc{\xi,\eta} \le x}=$$
$$=pF(x) + (1-p)\cdot\Pb\hc{\xi \le x,\; \eta \le x} = pF(x) + (1-p)\cdot\Pb\hc{\xi \le x}\cdot\Pb\hc{\eta \le x} = pF(x) + (1-p)F(x)G(x).$$
Все осуществлённые здесь логические переходы корректны в силу независимости случайных величин.
\end{solution}

\begin{problem}
Доказать, что если $\xi$ имеет геометрическое распределение, то имеет место равенство
$$\Pb\hc{\xi = n+k\;|\;\xi \ge k} = \Pb\hc{\xi = n},\; k,n = \hc{0,1,2,\dots}.$$
\end{problem}
\begin{solution}
Если величина имеет геометрическое распределение, то $\Pb\hc{\xi = k} = p^k(1-p)$. Легко видеть, что
$\Pb\hc{\xi \ge k} = \suml{i=k}{\infty}\Pb\hc{\xi = i} = (1-p)p^k(1 + p + p^2 + \dots) = (1-p)\frac{p^k}{1-p}= p^k$.
С другой стороны, $\Pb\hc{\xi = n+k\;|\;\xi \ge k} \bw= \frac{\Pb\hc{\xi = n+k,\; \xi \ge k}}{\Pb\hc{\xi \ge k}}$.
Далее, очевидно, что $\Pb\hc{\xi = n+k,\; \xi \ge k} = \Pb\hc{\xi = n+k,\; n+k \ge k} = \Pb\hc{\xi = n+k,\; n\ge 0} \bw=
\Pb\hc{\xi = n+k} \bw= p^{k+n}(1-p)$. Осталось подставить полученный результат в дробь: $p^k$ в числителе и знаменателе сокращается
и остаётся выражение $p^n(1-p)$, но это как раз и есть $\Pb\bc{\xi = n}$.
\end{solution}

\begin{problem}
Известно, что функция $F(s)$~--- производящая функция случайной величины $\xi$, которая принимает значения $\hc{0,1,2,\dots}$.
Найти производящую функцию последовательности $q_n := \Pb\hc{\xi \le n}$.
\end{problem}
\begin{solution}
Как мы знаем, производящая функция~--- это ряд $F(s) = \sumkzi s^k p_k$, где $p_k := \Pb\hc{\xi = k}$. Очевидно,
что $q_n = p_0 \spl p_n$, а найти нам надо $G(s) = \sumnzi s^n q_n = s^0(p_0) + s^1(p_0 + p_1) + s^2(p_0 + p_1 + p_2) + \dots$.
Поскольку наш ряд абсолютно сходится, применима теорема о сходимости двойного ряда, поэтому мы можем суммировать его как хотим.
Просуммируем <<по столбцам>>, собирая в одно слагаемое члены с одинаковыми индексами $p$. Получим:
$G(s) = p_0(s^0 + s^1 + s^2 + \dots) + p_1(s^1 + s^2 + s^3 + \dots) + p_2(s^2 + s^3 + s^4 + \dots) + \dots \bw=
\frac{p_0}{(1-s)} + \frac{p_1s^1}{(1-s)} + \frac{p_2s^2}{(1-s)} + \dots = \frac{1}{(1-s)}\sumkzi s^kp_k = \frac{F(s)}{(1-s)}$.
\end{solution}

\begin{problem}
Известно, что $\xi, \eta$~--- независимые случайные величины, распределённые показательно с параметром $\al$. Найти функцию распределения
и плотность случайной величины $\max\hc{\eta,\xi^3}$.
\end{problem}
\begin{solution}
По определению показательного распределения, $p_\xi = p_\eta = \al e^{-\al x}$, а $F_\xi = F_\eta = 1-e^{-\al x}$.
Пусть $G(x)$~--- искомая функция распределения. Тогда $G(x) = \Pb \bc{\min\hc{\eta,\xi^3} \le x} =
\Pb\bc{\eta \le x\; \vee\; \xi^3 \le x} = \Pb\hc{\eta \le x} \bw+ \Pb\hc{\xi \le \sqrt[3]{x}} - \Pb\hc{\eta \le x, \xi \le \sqrt[3]{x}}=
F(x) + F\hr{x^{1/3}} - F(x)F\hr{x^{1/3}}$. Автор предполагает, что проницательный читатель сам подставит вместо $F$ соответствующую
функцию и упростит выражения. Далее, чтобы получить плотность, надо этот результат аккуратно продифференцировать по $x$.
\end{solution}
