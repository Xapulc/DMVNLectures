\section{Начала теории оценивания}
\begin{nums}{-2}
\item Понятие статистической модели. Примеры: выборка, линейная гауссовская модель.
\item Теорема Гливенко.
\item Некоторые понятия теории оценивания: функции ущерба и риска, допустимые и байесовские оценки.
Несмещённое оценивание, квадратичный риск.
\item Неравенство информации (Крамера\ч Рао) для регулярных однопараметрических семейств: непрерывные
и дискретные распределения.
\item Эффективные оценки, экспоненциальные семейства распределений.
\item Многомерное неравенство Крамера\ч Рао.
\item Достаточные статистики: определение, примеры.
\item Достаточные статистики в случае нормальной выборки. Выборочные коэффициенты асимметрии и эксцесса.
\item Теорема факторизации (доказательства для элементарных случаев).
\item Улучшение несмещённых оценок путём их усреднения по достаточным статистикам: одномерная теорема Блеквелла\ч Рао.
\item Многомерная теорема Блеквелла\ч Рао.
\item Полные достаточные статистики: определение, примеры, единственность наилучшей несмещённой оценки.

\section{Условные математические ожидания и условные вероятности}

\item Напоминания: вероятностные пространства, случайные величины, абсолютная непрерывность мер,
производная Радона\ч Никодима.
\item Условное математическое ожидание случайной величины относительно $\si$\д алгебры: определение.
\item Некоторые простейшие свойства условного математического ожидания.
\item Условные вероятности, условные распределения.
\item Специальный случай: $\Ef(X|Y)$ для простых случайных величин $X$ и $Y$.
\item Некоторые дальнейшие свойства математических ожиданий. В частности:
$\Ef(\ph(Y)X|Y) = \ph(Y)\Ef(X|Y)$; условная дисперсия,
наилучший квадратичный прогноз.
\item Примеры вычисления $\Ef(X|Y)$.

\section{Оценивание в линейной модели}

\item Регрессионные и факторные (с одним и двумя факторами) линейные модели.
\item Достаточные статистики в линейной гауссовской модели.
\item Лемма об ортогональных разложениях случайного гауссовского вектора. Распределения $\chi^2$
(центральные и нецентральные).
\item Наилучшие несмещенные оценки параметров в линейной гауссовской модели и их
распределения.
\item Вычисление оценок наименьших квадратов в модели линейной регрессии.
\item Интервальные оценки для параметров нормальной выборки. Распределение Стьюдента (центральное и нецентральное).
\item Доверительные границы для вероятности успеха в испытаниях Бернулли.
\item Доверительные эллипсоиды для параметров линейной гауссовской модели. $\Fb$\д отношения и $\Fb$\д распределения.

\section{Проверка статистических гипотез}

\item Проверка статистических гипотез: общие принципы, и основные понятия (критическое множество, уровень значимости,
альтернативы, ошибки первого и второго родов, функция мощности).
\item Лемма Неймана\ч Пирсона.
\item Понятие о равномерно наиболее мощных критериях и пример: проверка гипотезы $\ta \le \ta_0$
против альтернативы $\ta > \ta_0$
по результатам испытаний Бернулли (здесь $\ta$\т вероятность успеха).
\item Связь между проверкой гипотез и доверительным оцениванием.
\item Проверка линейных гипотез в гауссовских линейных моделях с помощью критерия отношения правдоподобий.
\item Однофакторный дисперсионный анализ: проверка нулевой гипотезы (о равенстве эффектов обработки).
\item Двухфакторный дисперсионный анализ (аддитивная модель, одно наблюдение в клетке): проверка нулевой гипотезы
(о равенстве эффектов обработки).
\item Две гауссовские выборки, могущие отличаться сдвигом: проверка гипотезы об их однородности. Доверительные интервалы
для параметра сдвига.
\item Ранги наблюдений. Статистика ранговых сумм $W_{m,n}$ (Wilcoxon) для проверки гипотезы об однородности двух выборок.
Вычисление $\Ef W_{m,n}$ и $\Df W_{m,n}$ при гипотезе.
\item Точечные и интервальные оценки для сдвига одной выборки относительно другой с помощью статистики ранговых сумм
Уилкоксона.
\item Статистика Манна\ч Уитни $U_{m,n}$ и её связь с $W_{m,n}$.
\item Теорема Слуцкого.
\item Теорема об асимптотической нормальности двухвыборочных $U$\д статистик.
\item Асимптотические распределения статистик Манна\ч Уитни $U_{m,n}$ и Уилкоксона $W_{m,n}$ при $m,n\ra\bes$.

\section{Асимптотические методы оценивания и проверка гипотез}
\item Метод наибольшего правдоподобия. Неравенства теории информации.
\item Состоятельность оценок наибольшего правдоподобия (выборка, одномерный параметр).
\item Асимптотическая нормальность оценок наибольшего правдоподобия (выборки из регулярных семейств распределений).
\item Устойчивость оценок, функции влияния и примеры.
\item Статистические функции фон~Мизеса, функции влияния, устойчивость оценок, их асимптотические
свойства (эвристический вывод).
\item $М$\д оценки и их функции влияния.
\item Критерий согласия К.\,Пирсона для простой гипотезы. Теорема К.\,Пирсона.
\item Асимптотические свойства $M$\д оценок (эвристический вывод)
\item Критерии типа Пирсона\ч Фишера для сложных гипотез (без доказательства). Проверка гипотезы
о независимости признаков по таблице сопряжённости.
