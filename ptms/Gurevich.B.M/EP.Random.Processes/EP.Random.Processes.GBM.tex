\begin{nums}{-1}
\item  Общее определение случайной функции и~случайного процесса. Теорема
Колмогорова о~существовании случайной функции с~заданной системой конечномерных распределений
(формулировка). Существование случайной функции c независимыми значениями и~заданными одномерными
распределениями (\cite[\S\,5.1]{v}, \cite[гл.\,1 \S1]{gs}).
\item  Существование гауссовского процесса с~заданными математическим
ожиданием и~ковариационной функцией. Определение и~доказательство существования винеровского
процесса (\cite{l}, \cite[\S5.1]{v}).
\item  Определение и~свойства ковариационной функции случайного процесса.
Критерии существования предела и~среднеквадратической непрерывности случайного процесса, выраженные
в~терминах ковариационной функции (\cite{l}, \cite[\S1.3, \S2.1]{v}).
\item  Среднеквадратическое интегрирование случайного процесса (аналог
интеграла Римана). Интегрируемость непрерывного процесса (\cite{l}).
\item  Ортогональная векторная мера и~интеграл скалярной функции по такой
мере. Cтохастический интеграл по ортогональной случайной мере как частный случай. Ортогональная
случайная мера, построенная по процессу с~ортогональными приращениями. Ортогональная случайная мера
на $\mathbb R$, построенная по винеровскому процессу (\cite{l}, \cite[гл.\,7 \S\S\,1--7]{bsh},
\cite[\S2.2]{v}, \cite[\S7]{k}).
\item  Случайные процессы, стационарные в~широком и~узком смысле; различие
этих двух понятий в~общем случае (примеры) и~совпадение\т в~гауссовском. Спектральное
представление ковариационнной функции, спектральная мера и~спектральная плотность процесса (теоремы
Бохнера\ч Хинчина и~Герглотца: формулировка) (\cite[\S4.1]{v}, \cite[гл.\,1 \S5]{gs}).
\item  Спектральное представление стационарного в~широком смысле случайного
процесса $\xi_t$ как изоморфизм пространства $L_2$ по спектральной мере и~пространства случайных
величин, натянутого на все $\xi_t$ (\cite[\S4.2]{v}).
\item  Представление стационарного в~широком смысле случайного процесса в~виде
интеграла по ортогональной случайной мере (\cite{l}, \cite[\S2.2]{v}).
\item  Эргодическая теорема для стационарного в~широком смысле случайного
процесса с~дискретным и~непрерывным временем (\cite{l}, \cite[гл.\,7 \S10]{bsh},
\cite[\S4.2]{v}).
\item  Условия применимости закона больших чисел к~стационарному в~широком
смысле случайному процессу, выраженные в~терминах спектральной меры (\cite{l},
\cite[\S4.1, \S4.2]{v}).
\item   Спектральное условие среднеквадаратической дифференцируемости
стационарного в~широком смысле случайного процесса $\xi_t$. Спектр процесса $\xi'_t$ (\cite{l},
\cite[\S4.2]{v}).
\item   Построение процесса Орнштейна\ч Уленбека с~помощью стохастического
интеграла по ортогональной случайной мере, отвечающей винеровскому процессу (\cite{l},
\cite[гл.\,8 \S13]{bsh}).
\item   Процесс, построенный по последовательности независимых показательно
распределенных случайных величин: условное распределение моментов скачка, лежащих на отрезке
$[0,t]$, при условии, что число скачков равно $n$ (\cite{l}).
\item   Процесс, построенный по последовательности независимых показательно
распределенных случайных величин как пуассоновский процесс (\cite{l}).
\item   Различные виды непрерывности случайного процесса; примеры,
иллюстрирующие их несовпадение. Критерий существования непрерывной модификации случайного процесса
на отрезке (\cite{l}, \cite[\S5.2]{v}).
\item   Теорема Колмогорова о~существовании непрерывной модификации случайного
процесса на отрезке. Непрерывная модификация винеровского процесса на полупрямой (\cite{l},
\cite[\S5.2]{v}, \cite[\S8]{k}).
\item   Вывод равенства $\liml{t\ra\bes}\frac{W_t}{t}=0$ ($W_t$ --- винеровский
процесс) из теоремы существования непрерывной модификации (\cite{l}).
\item   Доказательство равенства $\uliml{t\ra\bes}\frac{W_t}{\sqrt t}=\infty$
($W_t$\т винеровский процесс). Формулировка закона повторного логарифма для $W_t$ (\cite{l},
\cite[гл.\,3 \S8, \S11]{bsh}, \cite[\S7.2]{v}).
\item   Среднеквадратическая сходимость суммы квадратов приращений
винеровского процесса на отрезке к~дли-\break не этого отрезка. Бесконечность вариации винеровского
процесса на отрезке. Недифференцируемость реализаций винеровского процесса (без доказательства)
(\cite[\S1.2]{v}, \cite[гл.\,3 \S1]{bsh}).
\item   Определение марковского момента и~момента остановки относительно
потока $\sigma$-алгебр. Момент достижения положительного уровня процессом с~непрерывными не
ограниченными сверху реализациями как момент остановки. Применение к~винеровскому процессу
(\cite{l}, \cite[гл.\,3 \S3]{bsh}).
\item   Принцип отражения для винеровского процесса (\cite{l}).
\item   Распределение момента первого достижения винеровским процессом
положительного уровня (применение принципа отражения) (\cite[гл.\,6 \S3]{gs}).
\item   Распределение максимума винеровского процесса на отрезке $[0,\,t]$
(применение принципа отражения) (\cite[гл.\,6 \S3]{gs}).
\item   Эквивалентность различных определений марковского процесса
(\cite{l}, \cite[гл.\,3 \S\S\,1--3, 5]{bsh}, \cite[\S8.2]{v}).
\item   Процесс с~независимыми приращениями как марковский процесс и~как мартингал (\cite{l},
\cite[гл.\,3 \S6]{bsh}).
\item   Переходная функция и~начальное распределение марковского процесса.
Уравнение Колмогорова\ч Чепмена и~его вероятностный смысл. Конечномерные распределения. Однородный
марковский процесс. Стационарное распределение (\cite[гл.\,6\,\S\S\,10--12]{bsh},
\cite[\S8.1 п.п. 1-5, \S8.2 п.п. 2, 4]{v}).
\item   Однородные марковские процессы с~дискретным множеством состояний.
Стандартные переходные функции. Дифференцируемость переходных вероятностей (формулировка).
Конечномерные распределения. Ма-\break трица интенсивностей. Мгновенные и~регулярные состояния.
Консервативные переходные функции (\cite[гл.\,6 \S12, \S16]{bsh}; \cite[гл.\,7 \S3]{gs}).
\item   Марковские процессы с~дискретным множеством состояний. Вывод обратных
уравнений Колмогорова (\cite[гл.\,6 \S17]{bsh}, \cite[гл.\,7 \S3]{gs}, \cite[гл.\,17 \S9]{f}).
\item   Марковские процессы с~дискретным множеством состояний.
Вывод прямых уравнений Колмогорова (\cite[гл.\,6 \S17]{bsh}, \cite[гл.\,7 \S3]{gs}, \cite[гл.\,17 \S9]{f}).
\end{nums}
