\section*{Билеты}

\begin{nums}{-3}
\item Эмпирическая функция распределения, её свойства как функции распределения и как оценки теоретической
функции распределения.
\item Теорема Гливенко\ч Кантелли о сходимости эмпирической функции распределения.
\item Асимптотические свойства эмпирических моментов и функций от них.
\item Теорема Колмогорова: доказательство независимости статистики Колмогорова от вида непрерывной функции
распределения.
\item Вариационный ряд, распределение порядковых статистик для равномерного семейства распределений.
\item Свойства частоты как оценки вероятности <<удачи>> в схеме испытаний Бернулли.
\item Условные математические ожидания и условные распределения относительно $\si$\д алгебр. Свойства
условных математических ожиданий.
\item Достаточные статистики. Теорема Неймана\ч Фишера (критерий достаточности).
\item Свойства статистических оценок. Оценки параметров нормального распределения.
\item Улучшение оценок с помощью достаточных статистик. Теорема Колмогорова\ч Блекуэлла\ч Рао.
\item Полные достаточные статистики и несмещённые оценки с минимальной дисперсией.
\item Эффективные оценки в регулярном случае. Неравенство информации (Крамера\ч Рао).
\item Информация Фишера и её свойства.
\item Метод моментов оценивания параметров. Состоятельность оценок метода моментов.
\item Метод максимального правдоподобия. Теорема об асимптотической нормальности оценок максимального правдоподобия.
\item Асимптотические свойства статистических оценок. Примеры состоятельных и асимптотически нормальных оценок.
\item Теорема о байесовской оценке при квадратичной функции риска. Априорный и апостериорный риск. Примеры
байесовских оценок.
\item Эквивалентность определений и свойства многомерного нормального распределения.
\item Лемма о независимости среднего арифметического и среднего квадратического для независимых одинаково
нормально распределённых случайных величин.
\item Распределения $\chi^2$, Стьюдента, Фишера\ч Снедекора как распределения статистик для нормальных
выборок.
\item Интервальные оценки. Построение точных доверительных интервалов.
\item Доверительные интервалы для параметров нормального распределения (для среднего при известной дисперсии
и для дисперсии при \textbf{неизвестном} среднем).
\item Доверительные интервалы для параметров нормального распределения (для среднего при \textbf{неизвестной} дисперсии
и для дисперсии при известном среднем).
\item Построение точного доверительного интервала для параметра биномиального распределения.
\item Асимптотические доверительные интервалы для параметра биномиального распределения.
\item Теорема Неймана\ч Пирсона. Критерий отношения правдоподобий для проверки двух простых гипотез, как
наиболее мощный критерий.
\item Равномерно наиболее мощные критерии. Свойства распределений с монотонным отношением правдоподобия.
\item Проверка гипотез о параметрах нормального распределения (для среднего при известной дисперсии
и для дисперсии при \textbf{неизвестном} среднем).
\item Проверка гипотез о параметрах нормального распределения (для среднего при \textbf{неизвестной} дисперсии
и для дисперсии при известном среднем).
\item Критерий Стьюдента равенства средних значений двух нормальных выборок.
\item Критерий Фишера равенства дисперсий двух нормальных выборок.
\item Дисперсионный анализ однофакторной модели для выборок из нормального распределения. Множественное
сравнение.
\item Теорема об асимптотическом $\chi^2$\д распределении статистики Пирсона.
\item Критерий $\chi^2$ для гипотезы о заданном распределении.
\item Проверка гипотезы  о значениях параметра биномиального распределения. Критерий знаков.
\item Критерий Колмогорова для проверки гипотез о данном непрерывном распределении.
\end{nums}

\section*{Программа экзамена}

\begin{items}{-3}
\item Основные понятия: выборка, статистическая модель, выборочные характеристики,
статистики; повторная выборка, параметрическая статистическая модель, функция правдоподобия.
\item Вариационный ряд, порядковые статистики, распределение порядковых статистик.
Квантили распределения и эмпирические квантили.
\item Статистические задачи для схемы Бернулли. Свойства частоты как оценки
вероятности удачи в схеме Бернулли. Критерии проверки гипотез о значении параметра биномиального
распределения. Критерий знаков.
\item Эмпирическая функция распределения, её свойства как функции распределения и
как случайного элемента (распределение и числовые характеристики). Сходимость эмпирической функции
распределения. Теорема Гливенко\ч Кантелли.
\item Теорема Колмогорова и доказательство независимости статистики Колмогорова от
вида непрерывной функции распределения. Критерий Колмогорова.
\item Условные математические ожидания и условные распределения относительно
сигма\д алгебр. Свойства ус-\break ловного математического ожидания. Аналоги формулы полной вероятности для условных
математических ожиданий и распределений.
\item Достаточные статистики. Теорема Неймана\ч Фишера (критерий достаточности).
Улучшение оценок с помощью достаточных статистик. Теорема Колмогорова\ч Блекуэла\ч Рао. Полные достаточные
статистики и их использование для нахождения несмещённых оценок с минимальной дисперсией.
\item Статистические оценки. Свойства оценок параметров в параметрической модели:
состоятельность, несмещённость, эффективность. Задача оптимального статистического оценивания.
\item Неравенство информации (Крамера\ч Рао). Эффективные оценки в регулярном
случае. Информация Фишера, её свойства.
\item Методы оценивания параметров. Метод моментов: свойство состоятельности
оценок. Метод максимального правдоподобия; свойство асимптотической нормальности и другие свойства оценок.
Оценки метода моментов и максимального правдоподобия для параметров нормального распределения, биномиального
и других распределений.
\item Асимптотические свойства статистических оценок: состоятельность и
асимптотическая нормальность. Состоятельность и асимптотическая нормальность функций от эмпирических
характеристик. Состоятельность и асимптотическая нормальность эмпирических моментов.
\item Байесовский подход к задачам статистического оценивания. Байесовские оценки
при квадратичной функции риска. Априорный и апостериорный риск, априорное и апостериорное распределение.
Построение байесовских оценок (для параметров биномиального и нормального распределений).
\item Нормальное распределение в $\R^n$, эквивалентность различных определений и
основные свойства. Распределение линейных и квадратичных форм от независимых нормальных случайных величин.
Независимость среднего арифметического и среднего квадратичного для независимых нормально распределённых
случайных величин. Распределения $\chi^2$, Стьюдента и Фишера\ч Снедекора как распределения статистик для
выборок из нормального распределения.
\item Интервальное оценивание параметров, доверительные интервалы. Построение
точных доверительных интервалов и асимптотических доверительных интервалов. Доверительные интервалы для
параметров нормального распределения. Точный и асимптотический доверительные интервалы для параметра
биномиального распределения.
\item Проверка гипотез о параметрах нормального распределения. Проверка
однородности двух нормальных выборок: критерий Фишера равенства дисперсий нормальных выборок, критерий
Стьюдента равенства средних значений нормальных выборок.
\item Однофакторная модель. Дисперсионный анализ $k$ выборок из нормального
распределения. Множественное сравнение.
\item Критерий $\chi^2$ для гипотезы о данном полиномиальном распределении.
Теорема об асимптотическом $\chi^2$\д распределении статистики Пирсона (теорема Пирсона). Критерий $\chi^2$
для проверки гипотезы о любом данном распределении.
\item Проверка статистических гипотез. Общие понятия: простые и сложные гипотезы,
статистический критерий, критическая область, вероятности ошибок I и II рода, размер и мощность критерия,
функция мощности критерия. Критерии согласия. Теорема Неймана\ч Пирсона: критерий отношения правдоподобия как
наиболее мощный критерий для проверки двух простых гипотез. Понятие равномерно наиболее мощного критерия.
Равномерно наиболее мощный критерий для семейства распределений с монотонным отношением правдоподобия.
\end{items}
